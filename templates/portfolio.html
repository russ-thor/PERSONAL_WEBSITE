Automated Stock Tracking

Python (m: Beautiful Soup, Pandas, Plotpy, EZGmail)

This project showcases pythons ability to automate many tasks. A Windows Task runs the program Mon-Fri after the stock market closes. The chosen tickers have their daily metrics scraped from Yahoo Finance with Requests/ Beautiful Soup and appended into a csv, a time series plot (both static + html) is created with Pandas and Plotpy, and the results are sent to the users email
